Change styles

This console is very customizable, you can change the style of the console by changing the values of the variables in the create event of the __debug_console__ object. The following variables are responsible for the style of the console:

Variable Description
console_width The width in pixels of the console.
console_height The height in pixels of the console.
console_bg_color The background color of the console.
console_bg_alpha The background alpha of the console.
console_text_font The font of the text in the console.
console_text_alpha The alpha of the text in the console.
console_text_blink_char The character that will blink when the console is waiting for input.
console_bar_height The height in pixels of the bar that shows the current command.
console_bar_color The color of the bar that shows the current command.
console_bar_max_chars The maximum number of characters that can be shown in the bar.
console_blur_amount The amount of blur that will be applied to the game behind the console.

Change key bindings

In the create event of the __debug_console__ object you can change the key bindings of the console by changing the values of the variables:

Variable Description Default
console_key_toggle The key that will toggle the console visible. F1
console_key_nav_up The key that will move the cursor up in the command history. vk_up
console_key_nav_down The key that will move the cursor down in the command history. vk_down

Change console's behavior

In the create event of the __debug_console__ object you can change the behavior of the console by changing the values of the variables:

Variable Description
console_suggestions_flag If this is set to true, the console will show suggestions when the user types a command.
console_blur_draw If this is set to true, the console will draw the game blurred behind it.
console_callback_on_open A function that will be called when the console is opened.
console_callback_on_close A function that will be called when the console is closed.
console_callback_on_log A function that will be called when a log is added to the console.
console_callback_on_destroy A function that will be called when the console is destroyed.
console_callback_on_game_end A function that will be called when the game ends.
